This is a node-simple-server example.

Overview

Creating a server

A simple server with an endpoint at http://localhost:8080/.

// Import the module const Server = require("node-simple-server"); // Create a new server, on port 8080 const localhost = new Server({ port: 8080 }); localhost.open(); // Open the server // Create an endpoint at root "/" localhost.endpoint("/", (request, result) => { result.writeHead("Content-Type", Server.mimeTypes.txt); // Set the content-type to text/plain result.write("Hello, World!"); // Write some text result.end(); // End the result });

Creating a file server

Setting a root directory, with an endpoint at http://localhost:8080/endpoint

/ Import the module const Server = require("node-simple-server"); // Create a new server, on port 8080 const localhost = new Server({ port: 8080, flags: [ Server.flags.FILESERVER ] customData: { rootDirectory: "./starting-folder/" } }); localhost.open(); // Open the server // Create an endpoint at "/endpoint" localhost.endpoint("/endpoint", (request, result) => { result.writeHead("Content-Type", Server.mimeTypes.txt); // Set the content-type to text/plain result.write("This is an endpoint on a fileserver"); // Write some text result.end(); // End the result });

Features

  • Automatic file extensions

    The URL http://localhost:3000/example is interpreted as http://localhost:3000/example.html, and includes a Content-Type: text/html header

  • Automatic mime-types

    When using Server.sendFile("./path/to/file", result), mime-types are automatically applied to the HTTP response
    All the accessable mime-types are located at node-simple-server/json/mime_types.json. If for whatever reason you would like to update this file, run node node-simple-server/mime_types-reload.js
